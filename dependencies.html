Dependencies¶

Conda-press currently has the following external dependencies,

Run Time:

  1. xonsh

  2. lazyasd

  3. ruamel.yaml

  4. tqdm

  5. requests

  6. conda (installed in the current environment)

All of these dependencies are available in conda-forge.
